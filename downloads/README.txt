MSS Keygen — Downloads folder
=============================

Put the file you want users to download in this folder.

The "Download Now" button links to:
  downloads/MSSKeygen.exe

So either:
  1. Name your file exactly "MSSKeygen.exe" and place it here, OR
  2. Change the href in index.html (look for: href="downloads/MSSKeygen.exe")
     to match your file name.

You can also update the version / size text in index.html
(look for the "download-meta" line).
